# Dotfiles for Ashwin

## Nuvio Collections
- **[Studios](/Studios/)** — Cover images used for Studio-based collections in [Nuvio](https://github.com/NuvioMedia/NuvioMobile), a streaming app that pairs with Debrid services (similar to Stremio) to deliver direct HTTP streams. This folder holds the studio logo artwork (20th Century, A24, Marvel, Pixar, Warner Bros., etc.), which maps each studio to its TMDB company ID so Nuvio can populate the collection.
